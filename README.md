# HomeLinq

**HomeLinq** is a UK property compliance and management platform built to help 
landlords, letting agents, and property managers track legal obligations, 
certifications, and tenancy compliance in one place.

## 🌐 Live Platform
[homelinq.co.uk](https://homelinq.co.uk)

## What it does
- Tracks property compliance deadlines (Gas Safety, EPC, EICR, etc.)
- Manages landlord and tenant documentation in line with gas safety and checks
- Sends automated renewal and expiry reminders to gas engineers, landlords and tenenats
- Provides a compliance dashboard for portfolio overview
